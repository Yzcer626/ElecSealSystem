## 1. 系统简介

电子印章系统是一个基于C++开发的汉字印章生成工具，支持从隶书字体文件生成BMP格式的印章图像。

### 1.1 主要功能

- **单字生成**: 将单个汉字转换为BMP图像
- **多字印章**: 支持横向或竖向多字印章
- **四字印章**: 支持传统四字印章（右上→左上→右下→左下）
- **状态管理**: 自动保存和恢复应用状态

### 1.2 系统要求

- **操作系统**: Windows 7/10/11
- **字体文件**: LiShu56.txt（隶书56x56点阵字体）
- **输出格式**: 24位真彩色BMP图像

- 多字印章：生成 `seal_multi_char.bmp`

## 3. 四字印章生成

### 3.1 什么是四字印章

四字印章是一种传统的印章形式，四个汉字按照特定顺序排列：

```
┌─────┬─────┐
│ 左上 │ 右上 │  <- 第1行
├─────┼─────┤
│ 左下 │ 右下 │  <- 第2行
└─────┴─────┘
```

**输入顺序说明**：
您需要按照 "右上→左上→右下→左下" 的顺序输入4个汉字。

例如，要生成 "春夏秋冬" 印章：
- 右上：春
- 左上：夏
- 右下：秋
- 左下：冬

所以应该输入：**夏春秋冬**（按照右上→左上→右下→左下的顺序）

### 3.2 生成四字印章

#### 命令行方式
```bash
seal.exe -4 "夏春秋冬"
```

或

```bash
seal.exe --four "夏春秋冬"
```

#### 交互式方式
```bash
seal.exe
```
然后选择四字印章模式，输入4个汉字。

### 3.3 常用四字印章示例

| 印章内容 | 输入顺序（右上→左上→右下→左下） | 命令 |
|---------|-------------------------------|------|
| 万事如意 | 事万意如 | `seal.exe -4 "事万意如"` |
| 福禄寿喜 | 禄福喜寿 | `seal.exe -4 "禄福喜寿"` |
| 恭喜发财 | 喜恭财发 | `seal.exe -4 "喜恭财发"` |
| 一帆风顺 | 帆一顺风 | `seal.exe -4 "帆一顺风"` |

### 3.4 输出文件

四字印章生成后，输出文件为：
```
bmp/seal_4char_diagonal.bmp
```

## 4. 单字/多字印章生成

### 4.1 单字模式（默认）

将每个汉字生成独立的BMP文件：

```bash
seal.exe "生日快乐"
```

输出：
- `bmp/UC9FA.bmp` (生)
- `bmp/UBFD5.bmp` (日)
- `bmp/UA5F6.bmp` (快)
- `bmp/UD6C0.bmp` (乐)

### 4.2 多字横向印章

将多个汉字横向排列生成一个印章：

```bash
seal.exe -m "万事如意"
```

或

```bash
seal.exe --multi "万事如意"
```

输出：
- `bmp/seal_multi_char.bmp`

### 4.3 多字竖向印章

（注：当前版本暂不支持命令行直接指定竖向排列，需要通过API调用实现）

## 5. 状态管理功能

### 5.1 自动保存

系统会自动保存以下状态：
- 窗口位置和大小
- 用户设置（输出目录、字体文件等）
- 最近使用的文件列表
- 最后使用的模式

### 5.2 手动保存状态

```bash
seal.exe --state-save
```

### 5.3 恢复状态

```bash
seal.exe --state-restore
```

恢复后会显示保存的状态信息：
- 版本号
- 窗口位置
- 窗口大小

### 5.4 清除状态

```bash
seal.exe --state-clear
```

清除所有保存的状态信息。

### 5.5 状态文件位置

状态文件保存在：
```
%LOCALAPPDATA%\ElecSealSystem\seal_app_state.dat
```

例如：
```
C:\Users\用户名\AppData\Local\ElecSealSystem\seal_app_state.dat
```

## 6. 命令行使用

### 6.1 命令格式

```
seal.exe [选项] [输入文本]
```

### 6.2 选项说明

| 选项 | 说明 |
|------|------|
| `-h`, `--help` | 显示帮助信息 |
| `-s`, `--single` | 单字模式（默认） |
| `-4`, `--four` | 四字印章模式 |
| `-m`, `--multi` | 多字印章模式 |
| `--state-save` | 手动保存应用状态 |
| `--state-restore` | 恢复应用状态 |
| `--state-clear` | 清除保存的状态 |

### 6.3 使用示例

#### 显示帮助
```bash
seal.exe -h
```

#### 生成单字印章
```bash
seal.exe "中"
```

#### 生成四字印章
```bash
seal.exe -4 "夏春秋冬"
```

#### 生成多字横向印章
```bash
seal.exe -m "生日快乐"
```

#### 保存当前状态
```bash
seal.exe --state-save
```

## 7. 常见问题

### Q1: 生成的图像无法查看？

**A**: 确保已安装支持BMP格式的图像查看器。Windows自带的照片应用即可查看。

### Q2: 命令行输入中文显示乱码？

**A**: 这是Windows控制台的编码问题。程序已使用宽字符API处理，不影响实际功能。建议使用交互式输入方式，或将命令保存为批处理文件后执行。

### Q3: 某些汉字无法生成？

**A**: 字体文件只包含GB2312字符集的7478个常用汉字，部分生僻字可能无法找到。请使用常用汉字进行测试。

### Q4: 四字印章的输入顺序是什么？

**A**: 四字印章的输入顺序为：右上→左上→右下→左下。

例如，要生成：
```
┌─────┬─────┐
│ 夏  │ 春  │
├─────┼─────┤
│ 冬  │ 秋  │
└─────┴─────┘
```

应该输入：`夏春秋冬`

### Q5: 如何修改输出颜色？

**A**: 当前版本通过命令行参数暂不支持自定义颜色。如需修改颜色，请编辑源代码中的颜色定义后重新编译。

默认颜色：
- 前景色（文字）：红色
- 背景色：白色
- 边框：红色

### Q6: 状态文件可以手动编辑吗？

**A**: 状态文件经过加密处理，不建议手动编辑。如需修改设置，请使用程序提供的功能或通过命令行参数。

### Q7: 如何备份状态？

**A**: 状态文件位于 `%LOCALAPPDATA%\ElecSealSystem\seal_app_state.dat`，可以直接复制该文件进行备份。

### Q8: 程序崩溃了，状态会丢失吗？

**A**: 不会。程序在正常运行时会定期自动保存状态，即使异常退出，大部分状态信息也能恢复。

### A. 支持的汉字范围

系统支持GB2312编码的汉字，包括：
- 一级汉字（常用字）：3755个
- 二级汉字（次常用字）：3008个
- 符号和图形：682个

总计：约7500个字符

### B. 输出文件命名规则

- **单字模式**: `U` + GB2312编码的十六进制（大写）+ `.bmp`
  - 例如：`中` → `UD6D0.bmp`
  
- **四字印章**: `seal_4char_diagonal.bmp`

- **多字印章**: `seal_multi_char.bmp`
